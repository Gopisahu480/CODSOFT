# Spam SMS Detection

## Objective
Build a machine learning model to classify SMS messages as Spam or Ham.

## Dataset
SMS Spam Collection Dataset

## Technologies Used
- Python
- Pandas
- Scikit-Learn
- Matplotlib
- Seaborn

## Machine Learning Algorithm
- TF-IDF Vectorizer
- Multinomial Naive Bayes

## Results
Accuracy: 97%+

## Project Structure
- Data Preprocessing
- Feature Extraction
- Model Training
- Evaluation
- Prediction
