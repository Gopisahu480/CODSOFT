# Credit Card Fraud Detection

## Objective

Build a machine learning model to detect fraudulent credit card transactions and classify them as Fraud or Legitimate.

## Dataset

Credit Card Fraud Detection Dataset

## Technologies Used

* Python
* Pandas
* NumPy
* Scikit-Learn
* Matplotlib
* Seaborn

## Machine Learning Algorithm

* Random Forest Classifier

## Results

Accuracy: 99%+

* Data Loading
* Data Preprocessing
* Feature Selection
* Model Training
* Fraud Prediction
* Model Evaluation
* Confusion Matrix Visualization
