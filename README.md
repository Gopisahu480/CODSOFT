# CODSOFT Machine Learning Internship

Welcome to my Machine Learning Internship repository completed under the CODSOFT Internship Program.

This repository contains the machine learning projects developed as part of the internship, demonstrating practical implementation of classification algorithms, data preprocessing, model training, evaluation, and prediction techniques using Python and Scikit-Learn.

* Internship Domain: Machine Learning
* Organization: CODSOFT
* Duration: Internship Program

---

## Completed Tasks

### Task 1: Spam SMS Detection

Developed a machine learning model capable of classifying SMS messages as Spam or Ham using Natural Language Processing (NLP) techniques.

#### Key Features

* Text preprocessing
* TF-IDF Vectorization
* Multinomial Naive Bayes Classifier
* Model Evaluation and Prediction

---

### Task 2: Credit Card Fraud Detection

Built a fraud detection system to identify fraudulent credit card transactions using machine learning classification techniques.

#### Key Features

* Data preprocessing
* Feature engineering
* Random Forest Classification
* Fraud prediction and evaluation

---

### Task 3: Customer Churn Prediction

Developed a predictive model to identify customers likely to discontinue a service based on customer demographics and account information.

#### Key Features

* Data preprocessing
* Label Encoding
* Random Forest Classification
* Churn prediction and analysis
